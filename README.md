# Maze Solver

A Java-based web application that visualizes maze solving using backtracking and pathfinding algorithms.

## Features
- Interactive maze grid
- Solve maze with backtracking
- Visual step-by-step tracing
- Simple web UI

## Run
1. Compile the Java sources:
   ```bash
   javac -d bin src/main/java/com/mazesolver/*.java
   ```
2. Run the application:
   ```bash
   java -cp bin com.mazesolver.MazeSolverApp
   ```

## Project Structure
- src/main/java/com/mazesolver/ - Java source files
- src/main/resources/web/ - HTML, CSS, and JavaScript for the UI
